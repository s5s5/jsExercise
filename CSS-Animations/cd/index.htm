成分简介

腾讯成都分公司2007年11月宣布成立。2008年5月，在四川地震后的十天，和成都市市政府正式签署投资总额5.5亿元的合作意向协议，也是灾后首家入川投资的企业。2008年7月正式注册成立腾讯科技（成都）有限公司，投资选址位于高新区天府软件园二期，并于2011年再次宣布追加投资额度至20亿元，包括购买园区C4、C5栋楼宇，以及位于开发区的50余亩科研用地，为腾讯在成都及西南的发展奠定了坚实的基础。

腾讯成都分公司目前开展的业务包括：大型网络游戏开发、手机应用及游戏产品开发、互联网及桌面软件开发、呼叫中心及在线客户服务、搜索技术服务、地方网络媒体及生活门户（大成网）、产品测试等领域，人员规模已经超过2000人。随着成都的经济发展及城市综合实力的提升，成都的人才吸引优势也越来越强，公司在成都的业务拓展也能更加有效地吸引西部人才，公司越来越多的BU及产品都将在成都拓展业务，成都分公司也将成为腾讯又一个非常重要的区域人才中心。

腾讯于2012年在成都天府新区已经启动建设两栋新的大楼，建筑面积超过20万平方米，预计2014年底初步建成，建成后将成为能够容纳万名员工的腾讯在中国西部最大的产品研发基地。
